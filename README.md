# anime-table

Таблица аниме тайтлов (~8000 записей)
Данные собраны из Kodik и Shikimori

Многие поля могут быть пустыми так как часть спаршенных даннных была лабо манга либо непопулярное аниме без даннных

---

## Поля

```json
{
  "id": 534,                         // внутренний ID
  "type": "movie",                   // тип (tv, movie, ova и т.д.)
  "year": "2017",                    // год выхода
  "title": "Баскетбол Куроко...",    // название (RU)
  "title_en": "Kuroko no Basket...", // название (EN)

  "status": "released",              // статус релиза
  "anime_status": "released",        // статус аниме
  "all_status": "released",          // общий статус

  "country": "Япония",               // страна
  "description": "...",              // описание
  "poster_url": "https://...",       // постер

  "anime_genres": "Спорт,...",       // жанры (строкой)
  "episodes_total": "1",             // количество эпизодов

  "shikimori_id": "31658",           // ID для Shikimori/Kodik
  "shikimori_rating": "8.08",        // рейтинг

  "released_at": "2017-03-18",       // дата выхода
  "createdAt": "2025-08-22T20:01:37.222", // дата создания
  "updatedAt": "2025-08-22T20:01:37.222"  // дата обновления
}
